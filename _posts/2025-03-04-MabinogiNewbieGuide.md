# 규칙

- 원활한 게임 진행을 위해서 

    - 본인 명의 계정 사용

    - 넥슨 OTP 가입

    - 넥슨 지정기기 등록(개인 PC일에서 게임 시작시 OTP 입력 불필요)

- 아르카나 개방 전까지

    - 6일 이상 지나 가능한 환생을 제외하고 비약을 사용한 환생은 하지 않습니다.(망각의 비약 포함)

- 스킬 수련 방법

    - 블로니의 지원 타이틀 + 종합 수련 스킬 포션 + 푸른빛 프리즘 사용한 상태를 기준으로 합니다.

- 퀘스트가 목록에 없거나 자동으로 오지 않는다면, 마을마다 있는 퀘스트 수주 게시판에 가서 찾을 수 있습니다.

- 금지 사항

    - 골드로 의장 구매하는 것을 지양합니다.

# 시작 ~ 블로니 (세 번째 추억담)

- `나오의 소개장` → `대륙의 정세` → `생활 스킬과 아르바이트` → `환생과 재능`
퀘스트를 완료하면 오는 `[성장지원] 바람을 타고 날아온 푸른 향기`를 통해 시작합니다.

### 메인 스트림

- 셰익스피어 - G13 햄릿 1-1 `비극의 음유시인` → **아본의 깃털** 획득

- 신의 기사단 - G19 신의 기사단 전체 → **실드 오브 트러스트** 획득

- 신의 기사단 - G20 성역의 문 전체 → **디바인 링크** 획득

- 아포칼립스 - G22 아포칼립스 전체 → **특성** 개화

    - 4-5 `눈을 감기는 자`를 완료 직후부터 [특성 한도 채우기](#특성-한도-채우기매주) 시작

    - 이후 오는 `[시간 왜곡]일렁이는 흔적` 퀘스트를 시작으로 `[급소 관통]극한의 일격`까지 진행하여 특성 획득

### 스킬 수련
    
1. **전투 점성술** 재능 마스터 달성

    - 퀘스트 수주 게시판에서 `[전투 점성술]별빛의 부름` 수행하여 관련 스킬 획득

    - `점성술과 별의 성채`까지만 진행하면 점투 점성술 스킬은 모두 얻을 수 있으며, 바로 수련을 시작하면 됩니다.

2. **마법사** 재능 그랜드 마스터 달성

3. **전사**, **체인 슬래셔** 중 1가지 선택하여 그랜드 마스터 달성
    
    - 전사 선택시 → 아르카나 엘레멘탈 나이트(그랜드 마스터 마검사)

    - 체인 슬래셔 선택시 →아르카나 다크메이지(그랜드 마스터 흑마도사)

        - 체인 슬래셔는 체인 무기를 착용하면 `룬스톤의 주인 찾기` 퀘스트를 시작할 수 있습니다.

        - [투아림 부스트 스킬 선행](#투아림-부스트-스킬-선행)을 먼저 시작하기 바랍니다.

## 평판(매일)

- `[가이드] 에린 주민들로부터의 평판` 수행

- 50레벨 달성까지 매일 진행해 줍니다.

    - 네일, 이보나 : 음유시인 게시판에 이보나 1분 평판 필사해서 연주 반복 

    - 브리아나 : 파티 가입후 다른 파티원이 음악 버프(전장의 서곡, 인내의 노래, 비바체, 행진곡) 공유

    - 엔델리온 : 밀 또는 보리 채집

    - 아이데른 : 레인 캐스팅 시전 된 상태에서 풍차에서 밀가루 또는 보릿가루 생산

### 특성 한도 채우기(매주)

- G22 아포칼립스 4-5 `눈을 감기는 자`를 완료하여 특성을 개화한 후 시작 가능합니다.

- 브리는 특성을 올릴 수 있는 일종의 포인트 입니다.

    - 매주 각 1500씩 획득 가능(프리시즌에는 3000), 보유 한도는 각각 5000

    - 매주 목요일 현실 시간 오전 7시에 한도 초기화

- 메인 스트림 진행을 통해 특성을 모두 얻어서 모든 레벨을 10을 올리기 전까지 매주 특정 작업을 반복 수행하여 브리를 얻어야 합니다.    

    - 수행의 브리 : 레벨업 총 150번

    - 도전의 브리 : 던전 클리어 총 30번

    - 교감의 브리 : 던바튼 글리니스 반복 퀘스트- **야채 페이스트** 총 30번 수행

### 투아림 부스트 스킬 선행

- `[투아림 부스트] 신뢰의 증거`에서 **견습 기사단 1명 호감도 레벨 5 달성**의 조건 달성에 필요합니다.

- 퀘스트 수주 게시판에서`[가이드] 알반 기사단 벨테인 특별조`를 받아 특별조 만들어 줍니다.(누적 레벨 500 이상)

    - 카나 또는 로간 영입 → 대화하기로 호감도 레벨 5 달성

    - 대화하기는 한번에 3번까지 가능하며, 게임 시간으로 오전 6시부터 다시 할 수 있습니다.

### 시드 피나하 입장 해금

- 퀘스트 수주 게시판에서`[콘텐츠]바람이 연주하는 야상곡`(누적 레벨 500 이상)을 받아 완료해 줍니다.

### 스타더스트 선행

- 퀘스트 수주 게시판에서`[가이드]왕립 천문학회의 새로운 발견`(누렙 3000이상)을 받아 완료해 줍니다.

- 이후에 오는 `[가이드]스타더스트의 코어와 제어구`도 완료해 줍니다.

### 펫 핸들러 재능

- 프라이스에게서 `핀 벨`을 구입 후 착용하면 오는 `[펫 핸들링]반짝이며 빛나는`퀘스트를 시작으로 순차적으로 완료 해 줍니다.

### [정령 무기](https://mabinogi.nexon.com/page/archive/guide_view.asp?id=4887434&num=43)

# 블로니 (네 번째 추억담) ~ 아르카나 최초 개방까지

### 메인 스트림

- 블로니 네 번째 추억담 전체(여신강림 - 드라마 시즌2)

- 아포칼립스 G23 템페스트

    - 클리어 후 오는 `깨어난 심해의 군주` 퀘스트까지 완료

- 이후 하지 않은 메인 스트림은 자유롭게 진행하면 됩니다.

### 스킬 수련

1. 그랜드 마스터 상인

2. 챔피언 모험가, 챔피언 펫 핸들러(핀즈 비즈 샤이닝 선행 작업)

3. 이후 모든 스킬 1랭, 그랜드 마스터 다 하시면 됩니다.

    - 아르카나 요구 재능, 자주 쓰이는 생활 관련(약사, 재단사, 운명 점성술사, 펫 핸들러, 대장장이, 목수) 위주로 우선 하는 것을 추천합니다.

    - 네. 어차피 전부 다 해야 해요. 

### [성장가이드](https://mabinogi.nexon.com/page/archive/guide_view.asp?id=4890845&num=18)

- 블로니 세 번째 추억담 완료 이후 던바튼 관청의 에반한테서 대표 캐릭터 등록 후 시작 가능합니다.

### 아르카나 개방

- `[가이드] 그랜드 마스터의 길` (누적 레벨 1만 이상)

    - 누적 레벨은 블로니 추억담을 통해서 올려주고, 모자란 부분은 아르카나 달성 미션을 통해 올려줍니다.

- 이후 `[가이드] 한 걸음 더 위로` 진행 → 아르카나 협회 레이르에게 `아르카나 적합자 소식` 물어보기

- 현재 아르카나의 링크를 5까지 개방 → 이후 다른 아르카나도 모두 5까지 개방해 줍니다.(보너스 스탯)

- 이후 본인이 주력으로 사용할 아르카나를 이후 10까지 링크를 개방합니다.

### 아르카나 달성 미션

- 블로니 세 번째 추억담 완료 이후 던바튼 관청의 에반한테서 대표 캐릭터 등록 후 시작 가능합니다.

- 최종 보상 `스페셜 위대한 재능 인장 선택 상자`에서 그랜드 마스터에 필요한 인장 9개를 한꺼번에 지급합니다.

    - 인장은 **음유시인**에 사용하는 걸 추천합니다.(아르카나 관련 재능만 선택 가능)

### 사도레이드 일일 지령 / 주간 지령(매일 또는 매주)

- G19 신의 기사단 클리어 후 오는 `아직 끝나지 않았다` 퀘스트 완료 후 매주 [주간/일일 지령](https://jump0whale.github.io/game/MabinogiApostleRaids.html) 수행

- 디바인 링크 서브 스킬 깨우침의 빛 15레벨 달성(경험치 45% 증가)

    - 이후 디바인 링크, 실드 오브 트러스트 나머지 서브 스킬 수련(신성 레벨 최소 150까지)

# 아르카나 개방 이후

## 레벨링 장소

※ **벨테인 - 배수의 진**을 이용한 레벨링은 여기서 다루지 않습니다.

- 연대기(주간 5회, 목요일 오전 7시 초기화)

    - 비 프리시즌에 6일에 한번 환생, 또는 매일 망각의 비약 1회 사용시에 추천합니다.

        - 망각의 비약 수급처 : 던전 가이드 보상, 던전가이드 상점, 성장 가이드 보상, 포텐셜, 이벤트 지급 등

    - 퀘스트 완료 보상 총 290% 이상 충족시 연대기 미션 1회로 레벨 200 달성 가능합니다.

        1. 프리미엄 팩 사용(+50%) + 응축된 아본의 크리스탈 이상 사용(240%)

        2. 또는 응축된 고급 아본 크리스탈(300%) 이상 단독으로 사용

    - [연대기 미션 공략](https://www.youtube.com/watch?v=y6TAzUJ_4680)

    - [연대기 미션 비밀요소 정리](https://arca.live/b/mabi/113906159?category=%EC%A0%95%EB%B3%B4%28%EC%A0%9C%EB%B3%B4%29&p=1)


- 테흐 두인
    
    - 환생의 비약이나 망각의 비약을 이용하여 2번이상 연속적인 레벨링을 할 때 추천합니다.

    - 선택지가 있지만, 접근성이 가장 좋은 심해의 깨어난 군주 어려움을 추천합니다.

    - 아래의 버프들을 필요합니다.

    - 필요한 갯수가 많을수록 레벨업이 더 빨라지기 때문에 특정 기간에만 얻을 수 있는 5, 6을 위해서 프리시즌에만 이용하는 걸 추천합니다. 

## 레벨링에 필요한 버프(테흐두인, 벨테인)

- 필수 

    1. 그랜드 마스터 사제(50%) + 펫 디바인 링크 상태(45%) + 그랜드 마스터 사제 파티(50%)

    2. 강화된 전투 경험치 포션(4배), 로열 전투 경험치 포션(5배), 라데카의 전투 경험치 포션(4배) 중 1

- 선택

    3. 노란빛 프리즘(경험치 2배)

    4. 행복을 기원하는 신비한 영약(경험치 3배, AP 3배)

- 기타

    5. 한정 타이틀(블로니의 특별 지원, 다시 돌아온 밀레시안)

    6. 이벤트 버프(경험치 2배 또는 3배)

## 불타래, 얼어붙은 불타래 수급 방법

- 불타래
    - 테흐두인 던전 어려움(파티 플레이 권장) 기본 보상과 더불어 추가로 드랍되는 테흐두인 보상상자에서 불타래를 선택하여 매주 고정적으로 수급

    - 또는 이벤트 통행증 사용이나 심연의 상급던전 클리어

- 얼어붙은 불타래 → 경매장에서 구매

- 스타더스트는 비 프리시즌에 진행되는 출석체크 이벤트를 이용해 얻는 지원 선택 상자를 모아 뒀다가 프리시즌에 시작하는 것을 추천합니다.

    - 지원 선택 상자에서 로열 아카데미 크리스탈과 프리시즌 기간 학회 보상 2배 중복 사용 가능(총 4배)
